# Opus: Verification Telegram Bot

Этот бот предназначен для ручной модерации анкет сотрудников.

Что умеет:
- получает новые анкеты от основного Telegram-бота;
- автоматически подписывает чат на новые анкеты после `/start`;
- показывает кнопки `Верифицировать` и `Отклонить`;
- умеет по команде `/pending` заново показать очередь анкет со статусом `pending`;
- записывает решение прямо в таблицу `employees` в Supabase.

Запуск:
1. `pip install -r requirements.txt`
2. Настройте `.env`
3. `python main.py`
