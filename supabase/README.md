# База данных и Миграции (Supabase)

Здесь хранится SQL структура базы данных PostgreSQL (используется на платформе Supabase).

## 🛠 Технологии
- PostgreSQL 15+
- Supabase RLS (Row Level Security) политики
- UUID & Timestamps

## 📂 Файлы и их назначение
- **`migrations/001_init.sql`**: Полный скрипт инициализации. 
  - Создает таблицы: `employers`, `employees`, `tariff_plans`, `subscriptions`, `card_views`, `payments`.
  - Включает **RLS (Row Level Security)**, чтобы пользователи могли видеть только свои данные.
  - Вставляет дефолтные тарифные планы при запуске скрипта.

## 💡 Для ИИ Агентов
Как запускать изменения в БД:
Проект использует облачный Supabase. Чтобы применить новую структуру (или восстановить базу):
1. Откройте панель SQL Editor в Dashboard Supabase.
2. Скопируйте содержимое `001_init.sql` и выполните.
*Внимание: скрипт очищает предыдущие таблицы (`DROP TABLE IF EXISTS ... CASCADE`), не запускайте на продакшене с живыми данными!*
