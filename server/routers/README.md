# Серверные Роутеры (FastAPI)

Эта папка содержит все эндпоинты (API маршруты) для взаимодействия фронтенда с бэкендом и базой данных Supabase.

## 🛠 Технологии
- FastAPI `APIRouter` для модульного разделения логики.
- `Depends` для внедрения зависимостей (например, проверка JWT токенов).

## 📂 Файлы и их назначение
1. **`auth.py`**: Отвечает за логин, регистрацию (работодателей) и сброс пароля. Работает напрямую с Supabase GoTrue Auth API. Использует `get_anon_client()`.
2. **`employees.py`**: Отвечает за кандидатов:
   - `GET /` и `GET /count`: Публичные маршруты без авторизации, возвращают базовую инфу для сетки карточек на главной фронтенда.
   - `POST /{id}/view`: Приватный маршрут (требует JWT). Проверяет подписку работодателя, списывает 1 "лимит" просмотра и возвращает полные контакты кандидата.
3. **`tariffs.py`**: Возвращает список доступных для покупки тарифов и текущую подписку авторизованного работодателя.
4. **`payments.py`**: Интеграция с платежной системой (в разработке Fenik Pay). Создает `payment_id` и перенаправляет на оплату.

## 💡 Для ИИ Агентов
Если нужно добавить новый эндпоинт, создайте его в соответствующем файле (или создайте новый файл и не забудьте подключить его через `app.include_router` в `main.py`). Соблюдайте принципы разделения: публичные роуты (без `Depends(get_current_user)`) и приватные (с ним).
