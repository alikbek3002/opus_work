# Opus: Веб-платформа работодателей (React + Vite)

## 📌 Назначение
Это фронтенд приложение, с которым взаимодействуют работодатели. Они регистрируются, авторизуются, могут оплачивать тарифные планы и просматривать список карточек сотрудников (кандидатов). 

## 🛠 Используемые технологии
- **Ядро:** React 19, TypeScript, Vite.
- **Роутинг:** `react-router-dom`.
- **Стэйт Менеджмент и Кэширование:** `@tanstack/react-query` v5+.
  - Query Client настроен на глобальное кэширование и предотвращение дублирования запросов.
  - Используются кастомные хуки в `/hooks` (`useAuth`, `useEmployees`, `useTariffs`).
- **Стилизация:** Tailwind CSS (`@tailwindcss/postcss`), Radix UI (shadcn компоненты). Компоненты адаптированы с помощью Framer Motion для плавных анимаций.
- **API Клиент:** Находится в `src/lib/api.ts` — централизованный обработчик запросов с автоматическим открытием LoginModal (через кастомные ивенты `openLoginModal`) при получении ошибки **401 Unauthorized**.
- **Auth Flow:** Вместо страниц `/login` и `/register` используются модальные окна, вызываемые "по требованию" при попытке просмотреть карточку неавторизованным пользователем.

## 📂 Архитектура
- `src/components/ui/` — Переиспользуемые "глупые" компоненты UI (из Radix / Shadcn).
- `src/components/` — Смарт компоненты (Модалки, Карточки). Важные: `LoginModal.tsx`, `EmployeeCard.tsx`.
- `src/pages/` — Страницы (Dashboard, Tariffs).
- `src/hooks/` — React Query хуки бизнес логики.
- `src/lib/api.ts` — Наш кастомный 'axios', который отправляет запросы на FastAPI (с Bearer токеном из localStorage).

## 🚀 Запуск для ИИ Агентов
1. Окружение `Node.js` (рекомендуется 20+).
2. `npm install` (Если проблемы с правами `npm` кэша, используйте `--cache /tmp/npm_cache` или сбросьте владельца директории `/Users/.../.npm` через `chown`).
3. Команда запуска: `npm run dev`.
4. В `.env` файле должен быть указан `VITE_API_URL` (например `http://localhost:8000/api`).
