# Opus: Telegram Bot (Сбор анкет сотрудников)

## 📌 Назначение
Этот модуль отвечает за сбор данных от соискателей/кандидатов. Сотрудник заходит в бота, пишет `/start` и проходит пошаговую регистрацию. В конце бот сохраняет все данные (ФИО, телефон, район, возраст, опыт и т.д.) в центральную базу данных **Supabase**, в таблицу `employees`.

## 🛠 Используемые технологии
- **Язык:** Python 3.10+
- **Библиотеки:** 
  - `python-telegram-bot` (v20+) — основной фреймворк для взаимодействия с API Telegram (использует `ConversationHandler` для пошагового сбора данных).
  - `supabase` — Python-клиент для взаимодействия с БД напрямую (service_role token).
  - `python-dotenv` — для чтения переменных окружения.

## 📂 Архитектура (папка `/bot`)
- `main.py` — Точка входа, инициализация приложения и хендлеров.
- `config.py` — Загрузка `.env` (Токены, ключи Supabase).
- `database.py` — Инициализация Supabase-клиента и CRUD-операции.
- `/handlers/`
  - `start.py` — Обработчик `/start` (Приветствие и проверка, зарегистрирован ли уже пользователь).
  - `registration.py` — Основной модуль с `ConversationHandler`. Пошагово запрашивает ФИО, телефон, район, возраст, опыт работы, портфолио и специализацию.

## 🚀 Запуск для ИИ Агентов
1. Установите зависимости: `pip install -r requirements.txt`
2. Настройте файл `.env`.
3. Запустите: `python main.py`

**Особенности логики:** Бот сам по себе не требует API-сервера для работы — он напрямую общается с Supabase (как микро-сервис).
